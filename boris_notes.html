Configuration pour changer l'icone de l'application 2 commandes sont obligatoires: - flutter pub add flutter_launcher_icons Cette commande ajoute flutter_launcher_icons à votre projet - flutter pub add flutter_native_splash Cette commande ajoute flutter_native_splash à votre projet Suivre la configuration actuelle du fichier .yaml ensuite pour signifier a flutter de prendre en compte les modifications flutter pub get Pour générer une nouvelle icone: - flutter pub run flutter_launcher_icons:main - flutter pub run flutter_native_splash:create
